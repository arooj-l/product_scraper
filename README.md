
#  Product Scraper – Books to Scrape

## Overview
This project is a Python-based web scraper that extracts product information from the practice e-commerce website **http://books.toscrape.com**. The scraper collects details such as book titles, prices, ratings, availability, and product links, and saves them into CSV files.

##  Features
- Scrapes product information from an e-commerce website.
- Handles pagination to extract data from all pages.
- Saves structured data into CSV files.
- Uses a virtual environment for dependency management.
- Clean and modular project structure.

##  Technologies Used
- **Python**
- **Requests** – for sending HTTP requests.
- **BeautifulSoup** – for parsing HTML and extracting data.
- **CSV** – for storing structured data.
- **VS Code** – development environment.
- **Git & GitHub** – version control and portfolio showcase.
